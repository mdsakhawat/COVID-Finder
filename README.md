# COVID-Finder

It's a simple android app for tracking all COVID19 patients around 100 meters from a user . The users who are COVID positive then their latitude and longitude are transfered to firebase real time database .Then all COVID positive users's locations are retrieved from firebase database and showed on map as a marker .Then users can see how many COVID positive users around 100 meters from the current user as a Toast .
